# Weather Update
***Overview***

This project is an automated weather page built using GitHub Pages and GitHub Actions.
It displays the current weather in Warsaw and updates automatically every hour without manual intervention.

***Data Source***

The project uses the Open-Meteo API to fetch real-time weather data, this API was chosen because:

- It is completely free to use

- No authentication or API key is required

- It provides reliable and up-to-date weather data

- Easy to integrate into automation workflows

***How to Run the Workflow Manually***

The workflow can be triggered manually following these steps:

1. Go to the Actions tab in the repository

2. Select the workflow named "Update Weather"

3. Click "Run workflow"

4. Confirm by clicking Run workflow
